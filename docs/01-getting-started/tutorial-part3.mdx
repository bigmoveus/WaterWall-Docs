# آموزش بخش سوم

اگر یادتان باشد در بخش اول ما فایل core.json را آماده کردیم و درون آن گفتیم که میخواهیم کانفیگ فایل config.json را اجرا کنیم

هم اکنون فایل config.json را ایجاد میکنیم و به ساختار زیر میرسیم


```
~/waterwall/
├── Waterwall          # فایل اجرایی برنامه
├── core.json          # تنظیمات اصلی
├── config.json        # فایل کانفیگ برای جرا
```



## ساختار کلی config file

این فایل در حالت کلی به شک زیر هست
