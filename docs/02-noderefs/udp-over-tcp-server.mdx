## عملکرد

توضیحات اصلی و سناریوهای استفاده در سند [UdpOverTcpClient](./udp-over-tcp-client) آمده است؛ لطفاً آن را مطالعه کنید تا با منطق کلی آشنا شوید. این نود در سمت مقابل قرار می‌گیرد و برای بازسازی مرز بسته‌های UDP روی جریان TCP استفاده می‌شود.

این نود باید همراه با `UdpOverTcpClient` استفاده شود.

این نود تنظیمات خاصی ندارد.

```json
{
    "name": "node_name",
    "type": "UdpOverTcpServer",
    "settings": {},
    "next": "next_node_name"
}
```

- این نود داده را تا دریافت کامل بسته‌هایی که توسط `UdpOverTcpClient` فریم شده‌اند بافر می‌کند و از ارسال بسته‌های ناقص UDP جلوگیری می‌کند.
- این نود ۲ بایت ابتدایی هر بسته (طول بسته) را که توسط `UdpOverTcpClient` اضافه شده است حذف می‌کند؛ این ۲ بایت بخشی از بسته اصلی UDP نیست.
